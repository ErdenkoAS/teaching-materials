# Технологии программирования

Курс читается в рамках следующих образовательных программ НИУ ВШЭ — Высшая Школа Бизнеса, 1 курс, 2025–2026:

- Бизнес-информатика
- Управление цифровым продуктом
- Технологии анализа данных в бизнесе

## Формула итоговой оценки

Final = 0.1 × Посещаемость + 0.15 × Активность + 0.05 × Тесты + 0.1 × ДЗ + 0.2 × Хакатон + 0.4 × Экзамен
- Активность = 0,45 × Лекции + 0,55 × Семинары
- Экзамен = очный, письменный, блокирующий
